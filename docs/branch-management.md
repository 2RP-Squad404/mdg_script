# **Gerenciamento de Branches**:

Padrão estabelecido:  
`novidade/nome-da-novidade`

Exemplos de branch:  
`feature/feature-name`  

----

Regras:

- Utilizar somente letras minúsculas;
- Não utilizar acentuação;
- No lugar dos espaços utilizar `-`;
- Inglês.
